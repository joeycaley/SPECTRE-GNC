# SPECTRE-GNC
Guidance, navigation, and control (GNC) analysis for CU Boulder's Spring 2026 ASEN 5148 Spacecraft Design course for the Moonraker mission. This was done for team M1, aka the South Pole Expedition for Circumpolar Terrain Reconnaissance and Exploration​ mission or SPECTRE.
